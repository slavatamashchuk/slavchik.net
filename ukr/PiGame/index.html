π гра

Що таке π гра?

π гра — гра зроблена за допомогою Arduino. Ціль гри — вчити цисло π. Гра підтримую 159 знаків після коми числа π.

Код проєкту

Код проєкту є на моєму github репозиторію по посиланню тут.

У проєкті є наліпка яку ви можите надркукувати з документу. Ви можите відкрити документ тут.
